# 🌙 fajr — Атмосферный виджет времени намаза
> **fajr** — современный веб-виджет для отслеживания времени намазов с уникальной атмосферной анимацией (дождь, снег, звёзды, облака, луна и др.). Интерфейс адаптивен, работает на любых устройствах, поддерживает PWA.

| Функция | Описание |
|---------|----------|
| 🕌 **Точное расписание** | Фаджр, Зухр, Аср, Магриб, Иша + Восход и Тахаджуд |
| ⏳ **Обратный отсчёт** | Динамическое обновление до ближайшего намаза |
| 🌤️ **12 погодных эффектов** | Дождь, снег, песчаная буря, облака, туман, звёзды, падающие звёзды, луна, солнечные лучи |
| 💨 **Скорость ветра** | Регулировка влияет на облака, дождь и песок |
| 🌓 **Фазы луны** | Реалистичное отображение в ночное время |
| 🎨 **Тёмная / светлая тема** | Переключается одним кликом |
| 💾 **Сохранение настроек** | Все эффекты и тема сохраняются в браузере |
